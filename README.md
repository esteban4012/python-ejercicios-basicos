# 🐍 Ejercicios y Proyectos en Python

Este repositorio contiene ejercicios y proyectos desarrollados en Python como parte de mi proceso de formación en desarrollo de software, enfocado en fortalecer la lógica de programación y prepararme para el mundo laboral.

---

## 📌 Contenido

### 🔹 Ejercicios básicos

* Programa para calcular el promedio de números ingresados por el usuario
* Validación de datos utilizando `try/except`
* Uso de listas, ciclos y estructuras condicionales

---

### 🔹 Proyecto: Sistema de Gestión de Usuarios (CRUD básico)

Aplicación en consola que permite gestionar usuarios con las siguientes funcionalidades:

* ✅ Registrar usuarios
* ✅ Validación de datos (evita nombres vacíos)
* ✅ Control de duplicados
* ✅ Listado de usuarios
* ✅ Eliminación de usuarios

---

## 🚀 Tecnologías utilizadas

* Python

---

## 🎯 Objetivo del proyecto

Desarrollar habilidades en lógica de programación, estructuras de control y buenas prácticas, con el objetivo de desempeñarme como desarrollador de software.

---

## ▶️ Cómo ejecutar el proyecto

1. Clonar este repositorio:

```bash
git clone https://github.com/esteban4012/python-ejercicios-basicos.git
```

2. Acceder a la carpeta del proyecto:

```bash
cd python-ejercicios-basicos
```

3. Ejecutar el programa:

```bash
python main.py
```
